# API reference

This section is the reference map for the APIs that power x402 Studio and Singularity Layer.

It covers the main surfaces already described across the guides:

- marketplace discovery
- public endpoint and product purchase flows
- webhook and receipt verification
- owner and agent management APIs
- MCP control plane entrypoints

## Auth modes

Different surfaces use different auth models:

| Surface | Auth |
| --- | --- |
| Marketplace discovery | none |
| Hosted product and endpoint purchase | x402 payment |
| Webhook delivery | HMAC signature verification |
| Receipt verification | JWT + JWKS |
| Owner management | PAT or legacy endpoint key |
| Wallet-first agent flows | signed wallet challenge → short-lived session token |
| MCP | PAT or endpoint key depending on tool |

## Reading this reference

- Start with **Marketplace discovery** if you need public lookup or listing data.
- Start with **Payments and checkout** if you need hosted or programmatic purchase flows.
- Start with **Webhooks and receipts** if you need post-payment verification.
- Start with **Owner and agent APIs** if you need authenticated management or agent registration.
